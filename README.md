# Auth-PHP

Ce projet est un système d'authentification **PHP simple, léger et sécurisé**, conçu pour fonctionner avec seulement **deux fichiers** :  
- `login.php`  
- `dashboard.php`

Il a été **réalisé dans le cadre d’un projet scolaire** pour apprendre et démontrer les bonnes pratiques d’authentification web sécurisée.
